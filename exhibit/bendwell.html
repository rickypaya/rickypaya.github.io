BendWell

This project was developed as part of the FullStack Academy's Capstone project in which teams of developers were given a week to strategize, implement, and deploy thier web application through various 3-tiered slices. The technical stack used for this project is a variation of the PERN (PostgresSQL, Express, React, Node.js) stack, wherein SupaBase acted in place of PostgresSQL and Express for the sake of time. The team first played with the idea of using pose-tracking technology, which led towards the theme of digital wellness for our application. The group then faced this task of diving the work into slices that made sense for the progression of the development of the app, which resulted in the individual stretches pages, the stretch routines and customization, and finally the guided session using pose-tracking to provide pose feedback. The team faced challenges such as deciding how the state of the web application would remember the order of stretches as they are played in the guided session, result in the use of a linked list data structure. The development team not only got an opportunity to build a web application using pose-tracking technology and training models, but we also gained insight into the itereative process of digital strategy and communication design as our own project evolved.
